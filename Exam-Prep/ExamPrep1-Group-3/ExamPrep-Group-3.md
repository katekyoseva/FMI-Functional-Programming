# Подготовка за Контролно 1 по Функционално Програмиране (2026)

Този списък съдържа 10 подбрани задачи, които покриват основните теми: рекурсия, съпоставяне на образци (pattern matching), функции от по-висок ред (`map`, `filter`, `fold`), композиция и работа с кортежи.

---

### Задача 1: Свиване на последователности (Compress)

Без да използвате готови функции, напишете рекурсивна функция `compress :: Eq a => [a] -> [a]`, която премахва съседни повтарящи се елементи.

- **Пример:** `compress [1,1,2,3,3,3,1] -> [1,2,3,1]`

### Задача 2: Проследяване на редувания (Sign Changes)

Използвайки `foldl` и кортеж за акумулатор, реализирайте функция `countSignChanges :: [Int] -> Int`. Тя трябва да преброи колко пъти се сменя знакът между два последователни елемента (напр. от положително към отрицателно число или обратно). Нулата не се счита за смяна.

- **Пример:** `countSignChanges [1, -2, -3, 4] -> 2`

### Задача 3: Ръчно разделяне на списък (Split)

Напишете функция `splitByPredicate :: (a -> Bool) -> [a] -> ([a], [a])`, която разделя даден списък на два: първият съдържа елементите, отговарящи на условието, а вторият – останалите. Реализирайте я чрез `foldr`, без да използвате вградената функция `partition`.

- **Пример:** `splitByPredicate even [1,2,3,4,5,6] -> ([2,4,6], [1,3,5])`

### Задача 4: Елегантна композиция (Function Pipeline)

Използвайки оператора за композиция `(.)` и стандартни функции от по-висок ред, напишете функция `countValid :: [Int] -> Int`, която:

1. Повдига всички числа на квадрат.
2. Изважда 10 от всяко.
3. Премахва отрицателните резултати.
4. Връща броя на останалите елементи.

### Задача 5: Съседни разлики (Adjacent Differences)

Използвайки функцията `zipWith`, напишете функция `adjacentDiffs :: Num a => [a] -> [a]`, която намира разликата между всеки два последователни елемента ($x_{i+1} - x_i$).

- **Пример:** `adjacentDiffs [1, 5, 2, 10] -> [4, -3, 8]`

### Задача 6: Честота на срещане (Frequency)

Напишете функция `frequency :: Eq a => [a] -> [(a, Int)]`, която приема списък и връща списък от кортежи, където всеки елемент е двойка `(елемент, брой_срещания)`.

- **Пример:** `frequency "banana" -> [('b',1),('a',3),('n',2)]`

### Задача 7: Намиране на индекси (Find Indices)

Напишете функция `findIndices :: (a -> Bool) -> [a] -> [Int]`, която връща списък от индексите на всички елементи, които отговарят на даден предикат. Опитайте се да я реализирате чрез помощна рекурсивна функция с акумулатор за текущия индекс.

- **Пример:** `findIndices (>3) [1, 5, 2, 6] -> [1, 3]`

### Задача 8: Вмъкване в сортиран списък (Insert Sorted)

Дефинирайте рекурсивна функция `insertSorted :: Ord a => a -> [a] -> [a]`, която вмъква елемент в предварително сортиран списък, така че крайният резултат да остане сортиран.

- **Пример:** `insertSorted 4 [1, 3, 5] -> [1, 3, 4, 5]`

### Задача 9: Безопасен достъп (Safe Access)

Дефинирайте функциите `safeHead :: [a] -> Maybe a` и `safeLast :: [a] -> Maybe a`. Те трябва да връщат `Nothing`, ако списъкът е празен, и съответния елемент, опакован в `Just`, ако списъкът не е празен.

- **Пример:** `safeHead [] -> Nothing`, `safeHead [1, 2] -> Just 1`

### Задача 10: Циклична ротация (Rotate)

Напишете функция `rotate :: Int -> [a] -> [a]`, която ротира списък наляво с $n$ позиции.

- **Пример:** `rotate 2 [1, 2, 3, 4, 5] -> [3, 4, 5, 1, 2]`
- **Подсказака:** Използвайте `take` и `drop`.
